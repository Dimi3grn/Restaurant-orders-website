# Recipe Manager

This is a React frontend application for managing recipes and orders. It uses a mock API service to simulate backend interactions.

## Features

- **Authentication**: User Login and Registration (Client & Admin roles).
- **Recipes**: Browse, Search, and Filter recipes.
- **Orders**: Clients can place orders for recipes.
- **Admin Dashboard**: Admins can Create, Update, and Delete recipes.

## Tech Stack

- React (Vite)
- TypeScript
- Tailwind CSS
- React Router DOM
- Lucide React (Icons)

## Setup & Run

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

## Mock Accounts

The app comes with pre-configured accounts (stored in LocalStorage on first load):

- **Admin**:
    - Username: `admin`
    - Password: `password`
- **Client**:
    - Username: `client`
    - Password: `password`
